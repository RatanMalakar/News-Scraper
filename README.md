# News Scraper Pipeline

## 🚀 Features
- Scrapes AI news from TechCrunch
- Cleans article content (removes ads/nav)
- Stores data in CSV 
- Backend API using Flask
- Frontend dashboard using React

## 🧠 Architecture
[Scraper] → [Data Cleaning] → [Storage] → [API] → [Frontend]

## ⚙️ Tech Stack
- Python (BeautifulSoup, requests)
- Pandas
- Flask
- React
